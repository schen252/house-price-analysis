# House Price Analysis

## Overview
This project analyzes a housing dataset to identify the key factors that influence property prices. The goal is to understand how different property features and external factors impact pricing.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Insights
- Property size (area) has a strong positive relationship with price.
- Homes in higher income-level areas tend to have significantly higher prices.
- Features such as additional bathrooms and security systems contribute to increased property value.

## Business Impact
These insights can help buyers, sellers, and real estate professionals make more informed pricing and investment decisions.
